HERALD

NAME

    herald - RSS-to-Email via SSH

COMMANDS

    ls              List uploaded feed configs
    cat <file>      Display config file contents
    rm <file>       Delete a config file
    run <file>      Manually trigger feed fetch and email
    logs            View recent delivery logs

EXAMPLES

    # Upload a config
    scp {{if ne .SSHPort 22}}-P {{.SSHPort}} {{end}}feeds.txt herald@{{.SSHHost}}:

    # Check status
    ssh {{if ne .SSHPort 22}}-p {{.SSHPort}} {{end}}herald@{{.SSHHost}} ls

    # View config
    ssh {{if ne .SSHPort 22}}-p {{.SSHPort}} {{end}}herald@{{.SSHHost}} cat feeds.txt

    # Manual run
    ssh {{if ne .SSHPort 22}}-p {{.SSHPort}} {{end}}herald@{{.SSHHost}} run feeds.txt
